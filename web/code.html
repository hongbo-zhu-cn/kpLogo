PKA: Positional Kmer Analysis

Limited capacity of the web server

This web server enables most options of PKA. However, to avoid use of excess computing resources, the length of the kmer is restrictd to 4 and users can only shuffle sequences preserving upto 4-nucleotide frequencies. Users can download the source code and install locally to run PKA without such restrictions.

Source code

The C++ source code for PKA is hosted on GitHub: https://github.com/xuebingwu/PKA .

Installation

Download the source code from github: https://github.com/xuebingwu/PKA/releases/
Unzip, enter the subfolder 'src', then type 'make':

cd src
make

The compiled programs are in the folder 'bin'.
To recompile, type 'make clean' then 'make'.
