PKA: Positional Kmer Analysis
PKA
Figure illustration: input -> PKA -> output

PKA allows you to:
1. discover and visualize well-positioned short motifs from aligned sequences
2. plot frequency logo, sequence (information) logo, probability logo, and kmer-logo at the same time
3. score sequences using positional motif information (commandline only)
