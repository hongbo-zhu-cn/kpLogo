PKA: positional kmer analysis (updated Dec 23. 2014)

This is a demonstration of the utility of the PKA tool sets. Input file size and search space will be limited on this server. Users can download the binary or source code to run at full capacity.

Reference: Wu et al, Detecting well-positioned short sequence motifs by positional k-mer analysis (PKA).
