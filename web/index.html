kpLogo: k-mer probability logo

k-mer probability logo (kpLogo) is a computational tool for sensitive discovery and visualization of position-specific ultra-short sequence motifs from a set of sequences with precisely defined positions.


Main features of kpLogo:

Compared to existing sequence logo generating tools, kpLogo is unique in several ways:
1. generates kmer logos that visualize posiiton interdependence and resolve overlapping motifs
2. handles weighted sequences or a ranked list of sequences
3. plots frequency logo, information content logo, probability logo, and kmer logo in one run
4. allow degenerate nucleotides and thus support gapped kmers
5. logos can be generated with colorblind friendly colors
